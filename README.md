W folderze "Literatura" jest sporo artykułów na temat epidemii palenia i książka z równań różniczkowych.

Proponuje wykorzystanie:
1.Elementary-Diffrential-Equation-and-Boundary-Value-Problem-Boyce-DiPrima - Stąd wziąłbym metody numeryczne takie jak Eulera i Algorytm Rungego-Kutty

2.Smoking epidemic model with density-dependent death rate and numerical sensitivity analysis - Tutaj mamy cały model epidemii palenia ze sporą ilością zmiennych i nawet jest on napisany fajnym językiem ale trzeba by było samemu ogarnąć, który parametr za co odpowiada (mój faworyt)

3.Analysis and Simulation of Fractional Order Smoking Epidemic Model - Tutaj jest paskudnie napisane, bo są tylko i wyłącznie wzory ale model jest ciekawie opisany

4.Raport z ogólnopolskiego badania ankietowego na temat postaw wobec palenia tytoniu - dane z polski z lat 2009-2019 co dwa lata, stąd najpewniej będziemy wyznaczać współczynniki lub sprawdzać równania 

5.Approximate Solution and Analysis of Smoking Epidemic Model with Caputo Fractional Derivatives - podobny do 2. ale ma mniej zmiennych i jest delikatnie prostszy, ma opis parametrów ale tutaj jest notacja D^(fi), której nie rozumiem
